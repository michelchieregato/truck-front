# Truck-front -> FrontEnd do projecto SIGA (Hackaton CCR)

Parte de front-end do projeto SIGA - aplicativo para caminhoneiros.

O backend desse projeto pode ser achado [neste repositório](https://github.com/michelchieregato/truck-back).

Essa parte do projeto é feita em NativeScript. Para instalação, sugere-se seguir os passos do [site oficial](https://nativescript.org/).

Ele é compilado para mac e ios (simular com `tns run android --bundle`, e `tns run ios --bundle`.

Porém, para testar a parte de mapas integrada, sugere-se compilar para android.
